# assets

Plaats hier het listing-icoon voor de GTM Community Template Gallery:

```
assets/icon.png
```

- Vierkant, 192×192 of 256×256 px, transparante achtergrond.
- HelloGekko-beeldmerk (geen kleine tekst).

Het icoon wordt vervolgens als base64 data-URI in `template.tpl` gezet
(`brand.thumbnail`); GTM bewaart het niet als los bestand.
